üîå API Endpoints

The system provides comprehensive REST APIs for all operations:

Employee APIs

GEThttp://localhost:8080/api/employees - Get all employees
POSThttp://localhost:8080/api/employees - Create new employee
PUThttp://localhost:8080/api/employees/{id} - Update employee
DELETEhttp://localhost:8080/api/employees/{id} - Delete employee

Payroll APIs

GEThttp://localhost:8080/api/payrolls - Get all payrolls
POSThttp://localhost:8080/api/payrolls/generate?employeeId=EMP001&payPeriod=2024-12 - Generate payroll
PUThttp://localhost:8080/api/payrolls/{id}/approve?approvedBy=admin - Approve payroll

Attendance APIs

POSThttp://localhost:8080/api/attendance/checkin?employeeId=EMP001 - Check-in
POSThttp://localhost:8080/api/attendance/checkout?employeeId=EMP001 - Check-out
GEThttp://localhost:8080/api/attendance/employee/{employeeId} - Get employee attendance

Leave APIs

POSThttp://localhost:8080/api/leaves/apply - Apply for leave
PUThttp://localhost:8080/api/leaves/{id}/approve?approvedBy=manager - Approve leave
GEThttp://localhost:8080/api/leaves/employee/{employeeId}/balance/{leaveType} - Get leave balance
